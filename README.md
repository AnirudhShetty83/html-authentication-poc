# Campus Pe assignment 1

This assignment contains 5 HTML pages demonstrating a basic authentication flow using plain HTML.
## Pages Included

- **login.html** – allows users to log in  
- **register.html** – allows new users to create an account  
- **forgot-password.html** – allows users to request a password reset  
- **reset-password.html** – allows users to set a new password  
- **dashboard.html** – represents the user dashboard after login
